# Pneumonia Detection from Chest X-rays using DenseNet121

## Project Overview

This project develops a deep learning-based medical image classification system for detecting pneumonia from chest X-ray images. The model classifies images into NORMAL and PNEUMONIA categories using the DenseNet121 transfer learning architecture.
